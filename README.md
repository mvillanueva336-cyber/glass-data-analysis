# Glass Data Analysis

Exploratory data analysis, feature engineering, and statistical modeling of the Glass Identification dataset using Python.

## Project Overview

This project analyzes the Glass Identification dataset using Python. The goal is to explore the data, perform preprocessing and transformations, evaluate feature relationships, and build statistical models to gain insights into the characteristics of different glass types.

## Techniques Used

* Train and test data splitting
* Stratified sampling
* Feature scaling with StandardScaler
* Box-Cox and Yeo-Johnson transformations
* Skewness analysis
* Correlation analysis
* Data visualization with Matplotlib
* Linear regression modeling

## Technologies

* Python
* Pandas
* NumPy
* Scikit-learn
* Matplotlib
* Statsmodels

## Dataset

Glass Identification Dataset from OpenML.
